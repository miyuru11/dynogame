🦖 Chrome Dino - Palm Controller
One Command Install & Play
Copy and paste this ONE command into Command Prompt:

cmd
cd C:\Users\%USERNAME%\Downloads\dino-palm-controller && python -m venv dino_env && dino_env\Scripts\activate && pip install mediapipe==0.10.9 opencv-python pyautogui numpy && python dino_controller.py
How to Play
Open Chrome → Type chrome://dino → Press SPACE

Show OPEN PALM to camera → Dino jumps!

Press q to quit

Need Python First?
Download Python 3.11: https://www.python.org/ftp/python/3.11.0/python-3.11.0-amd64.exe

⚠️ During install: CHECK "Add Python to PATH"

Troubleshooting
Issue	Fix
Dino won't jump	Click on Chrome window first
Camera error	Close Zoom/Teams
Python error	Install Python 3.11 above
