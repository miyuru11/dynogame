# 🦖 Chrome Dino - Palm Controller
This dino game is controlled by your hand.
 
  First do this 
## One Command Install & Play

Copy and paste this ONE command into Command Prompt:

```cmd
cd C:\Users\%USERNAME%\Downloads\dynogame && python -m venv dino_env && dino_env\Scripts\activate && pip install mediapipe==0.10.9 opencv-python pyautogui numpy && python dino_controller.py  ```

You Control the Game With Your HAND!
Your Hand	What Happens
🖐️ OPEN PALM (all fingers spread out)	Dino JUMPS!
✊ CLOSED FIST or any other hand position	Dino runs normally
No keyboard needed! Just show your palm to the camera!

How to Play
Open Chrome → Type chrome://dino → Press SPACE to start

Make sure Chrome window is ACTIVE (click on it)

Show your OPEN PALM to your webcam

Watch the dino jump!

Press q to quit the controller

Need Python First?
Download Python 3.11: https://www.python.org/ftp/python/3.11.0/python-3.11.0-amd64.exe

⚠️ During install: CHECK "Add Python to PATH"
